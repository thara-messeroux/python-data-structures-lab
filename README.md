# Python for JavaScript Developers: Data Structures Lab

Welcome to the **Python Data Structures Lab**! This repository contains a series of practical exercises designed to bridge the gap between JavaScript and Python by exploring Python's core data structures, including lists, tuples, dictionaries, and iteration techniques.

## 🚀 Objectives
By completing this lab, you will practice:
* Creating and manipulating **Lists** and **Tuples**.
* Utilizing **Dictionaries** for complex, key-value data storage.
* Iterating over multi-element data structures using `for` loops.
* Applying Python’s **slicing** and **indexing** features to slice and access data cleanly.

---

## 🛠️ Getting Started

### 1. Setup Your File
All exercises should be written inside a single Python script file named `exercises.py` in the root of this project directory.

### 2. Running Your Code
After completing an exercise or a specific function, run the script in your terminal to see the printed outputs and verify your results:
```bash
python3 exercises.py
